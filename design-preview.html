디자인 시안 미리보기

디자인 점수 87 → 88~89 진입을 위한 컴포넌트 상태 명확화 시안. 본체 사이트는 안 건드린 별도 페이지. 각 항목 왼쪽 현재 / 오른쪽 제안 비교. 우측 상단 ☽ 로 다크 모드 토글. 마음에 드는 항목만 본체 적용 PR 로 분리해서 진행.

1. 필터 칩 활성 상태 강화

현재도 활성 시 배경·색 반전이 적용돼 보여요. 다만 활성/비활성 구분이 사진이 많은 컨텐츠 사이에서 가끔 묻혀요. 제안: 좌측 ✓ 마이크로 표식 + inner ring 으로 시각 무게 +1.

현재

전체 컬러 흑백 슬라이드

활성 표시: 배경 채움 + 색 반전.

제안

전체 컬러 흑백 슬라이드

활성 표시: 배경 채움 + ✓ 표식 + inner ring 으로 강조.

2. Input focus ring 통일

현재는 페이지마다 input :focus 가 border-color 만 살짝 진해지는 정도라 모바일에서 잘 안 보여요. 제안: focus 시 border + 외곽 box-shadow ring 으로 키보드 사용자 / 모바일 양쪽 가시성 +.

현재

포커스 시 border 색만 진해짐. 미세함.

제안

포커스 시 외곽 ring 으로 명확. WCAG focus visible 충족.

3. 버튼 disabled 상태 명확화

현재 disabled 버튼이 일부 페이지에서 클릭 가능한 모양 그대로 유지돼 사용자가 무반응에 당황해요. 제안: opacity 0.4 + cursor:not-allowed + 색 약화로 즉시 분간.

현재

disabled 가 평소와 차이 거의 없음.

제안

disabled 가 즉시 분간. 커서도 not-allowed.

4. 스켈레톤 shimmer

현재 스켈레톤은 정적 회색 블록. 로딩이 멈춘 건지 진행 중인 건지 헷갈려요. 제안: 좌→우 shimmer 로 "지금 받고 있어요" 신호. prefers-reduced-motion 자동 비활성.

현재

정적. 멈춘 것처럼 보일 수 있음.

제안

좌→우 shimmer. 로딩 진행감.

5. 메타 텍스트 대비 보강 (옵션)

현재 --text-muted = rgba(0,0,0,0.65) 가 흰 배경 작은 폰트에서 WCAG AA 4.5:1 경계선이에요. 제안: 0.72 (다크모드 0.78) 로 살짝 진하게. 시각 차이는 미세하지만 명도 대비 안전 마진 확보.

현재

2026.06.01 · 박순렬 · 5분 읽기 · Comments 12

rgba 0.65.

제안

2026.06.01 · 박순렬 · 5분 읽기 · Comments 12

rgba 0.72 (다크모드 0.78).
