A. 타이포그래피 시스템 (Modular Scale 1.25)

현재 사이즈는 정성적 결정 (h1 36 / h2 22 / body 19 / meta 14). 비율 일관성 없음. 제안: 1.25 비율 modular scale (32 / 24 / 19 / 16 / 13). line-height 도 위계별 정교화 (heading 1.15~1.25, body 1.65~1.7). 매거진 가독성 + 시각 리듬 ↑.

B. 컬러 시스템 (Surface 계층 + Semantic 추가)

현재 토큰 6개 (text / text-muted / bg / bg-sub / border / accent). 위계 표현 한계. 제안: text 3 단계, surface 3 단계, border 2 종, semantic state (success / danger) 추가. 매거진 카드·모달·툴팁의 깊이감 명확.

현재 (6 토큰)

text#000
text-muted0.72α
bg#fff
bg-sub#f4f4f4
border#e0e0e0
accent#FFE600

위계 두 단계 (bg / bg-sub). 매거진 카드 위 카드 같은 3 단계 표현 불가.

C. 에디토리얼 시각 위계

현재 글 페이지는 본문 위주의 평면 구성. 매거진 정체성을 시각적으로 드러내는 요소 (kicker / lead / dropcap / pull-quote) 가 부재. 제안: 4 가지 매거진 컴포넌트로 글의 호흡 + 시각 리듬 + 매거진 톤 강화.
